Quality Standards & Codes

AWS D1.1  ·  ASME Boiler & Pressure Vessel Code  ·  ASTM Standard Numbering

Engineering materials and fabricated structures must meet defined quality standards to ensure safety, reliability, and interoperability. Three organizations dominate materials and fabrication standards in the United States: the American Welding Society (AWS), which governs structural welding practice; ASME (American Society of Mechanical Engineers), whose Boiler and Pressure Vessel Code (BPVC) covers pressure-critical equipment design and fabrication; and ASTM International, which defines material specifications used across virtually every engineering discipline. These standards are deeply interlocked — ASTM defines the material, ASME specifies how to build with it under pressure, and AWS governs how it is welded.

AWS — American Welding Society
Founded 1919 · Miami, FL · Publishes AWS D1.1 and the AWS A5.xx filler metal classification standards

AWS D1.1 Structural Welding Code — Steel

AWS D1.1 is the governing code for welded steel structures in the United States. It specifies design, fabrication, qualification, and inspection requirements for statically and cyclically loaded structures — including buildings, bridges, and equipment frames. Compliance is typically required by the International Building Code (IBC) and by contract documents on construction and heavy fabrication projects.

Code Structure — Major Clauses of AWS D1.1

Clause Title Key Content
1 General Requirements Scope, definitions, base metal limitations, responsibilities of Engineer and Inspector
2 Design of Welded Connections Joint types, effective weld sizes, allowable stresses, minimum fillet weld sizes by plate thickness
3 Prequalification of WPSs Prequalified joint details and base/filler metal combinations that do not require procedure testing
4 Qualification WPS qualification by testing (PQR); welder and welding operator performance qualification (WQT)
5 Fabrication Base metal preparation, assembly tolerances, preheat and interpass temperature, PWHT, weld repairs, arc strikes
6 Inspection Inspector responsibilities, visual inspection criteria, NDT methods, acceptance/rejection criteria
7 Stud Welding Stud base materials, equipment, qualification testing, and inspection requirements
8 Strengthening and Repairing Existing Structures Requirements for welded modifications and repairs to in-service structures

Welding Procedure and Welder Qualification Documents

WPS
Welding Procedure Specification
  • Written document stating all essential variables for a weld (process, position, base metal, filler metal, preheat, heat input, PWHT)
  • Must be followed by all production welders
  • Can be prequalified (no test coupon needed) or qualified by testing (requires a PQR)
PQR
Procedure Qualification Record
  • Test record that supports a non-prequalified WPS
  • Documents actual parameters used when welding a test coupon
  • Coupon is mechanically tested (bend, tensile, macro) to verify weld soundness
  • Filed and maintained by the fabricator — not required for prequalified WPSs
WQT
Welder Qualification Test
  • Certifies an individual welder's skill for a specific process and position
  • Also called Welder Performance Qualification (WPQ)
  • Test coupon examined by visual inspection and bend testing
  • Qualification typically expires after 6 months without use of the qualified process

Prequalified vs. Qualified-by-Testing WPSs

Criterion Prequalified WPS (Clause 3) Qualified by Testing — requires PQR (Clause 4)
Joint geometry Must exactly match D1.1 prequalified joint details Any geometry — proven by coupon testing
Base metal D1.1 Table 3.1 approved steels only (e.g., A36, A572, A992) Any steel, including unlisted or higher-strength grades
Process SMAW, SAW, GMAW (spray/pulse — not short-circuit), FCAW Any process including GTAW and short-circuit GMAW
Testing required None — but strict parameter limits must be maintained Mandatory mechanical testing, often radiographic or macro examination
Documentation WPS only (must cite the D1.1 clause being followed) WPS + one or more supporting PQR(s)

NDT Methods Recognized by AWS D1.1 (Clause 6)

Method Code What It Detects Typical Use in D1.1
Visual Testing VT Surface cracks, porosity, undercut, overlap, weld size and profile Required on all welds before any other NDT method is applied
Radiographic Testing RT Volumetric — porosity, slag inclusions, lack of fusion, cracks CJP groove welds in statically and cyclically loaded structures
Ultrasonic Testing UT Volumetric — planar flaws (cracks, lack of fusion); superior in thick plate Preferred for plate thickness > 8 mm (5/16″); may substitute for RT
Magnetic Particle Testing MT Surface and near-surface — cracks, seams, lack of fusion Ferromagnetic materials only; used for surface crack detection
Liquid Penetrant Testing PT Open surface discontinuities — cracks, porosity, laps Non-ferromagnetic materials (austenitic SS, aluminum) where MT cannot be used
Acceptance criteria depend on loading type. Statically loaded structures (Clause 6.9) use more permissive visual, RT, and UT acceptance limits than cyclically loaded structures (Clause 6.12), which face fatigue risk. Always verify which clause governs before evaluating an indication.
ASME — American Society of Mechanical Engineers
Founded 1880 · Boiler and Pressure Vessel Code first issued 1914 · Updated on 2-year revision cycles

ASME Boiler and Pressure Vessel Code (BPVC)

The ASME BPVC is the governing standard for the design, fabrication, inspection, and testing of boilers, pressure vessels, and nuclear components. It is enforceable by law in most U.S. states and Canadian provinces. Compliance is shown by an ASME Code Stamp stamped onto the component nameplate (e.g., "U" for an unfired pressure vessel built to Section VIII Division 1).

Section II: SA- vs. A- Material Designations

ASME Section II adopts ASTM material specifications with an "S" prefix. The SA- designation means the material meets the ASTM standard plus any additional ASME requirements — typically mandatory supplemental requirements for impact testing traceability and heat treatment records. Only SA- (or SB-, SFA-) certified materials may be used in ASME Code construction; an ASTM mill certificate alone is not sufficient.

ASTM International
Founded 1898 · Publishes over 12,000 consensus standards in the Annual Book of ASTM Standards (100+ volumes)

ASTM Standard Numbering System

ASTM International publishes consensus standards for materials, products, systems, and services. In materials engineering, ASTM standards define chemical composition, mechanical property requirements, permissible test methods, and required certifications for metals, polymers, composites, ceramics, and more. ASTM standards are widely adopted by purchase specifications, building codes, and the ASME BPVC.

How to Read an ASTM Designation

A
Letter prefix (material class)
·
572
Sequential standard number
–
21
Year of last revision
,
Gr. 50
Grade or class within the standard

Example: A572/A572M–21, Grade 50 — Ferrous (A), standard #572, revised 2021, Grade 50 (Fy = 345 MPa / 50 ksi minimum yield). A metric companion standard carries the suffix M (e.g., A572M); when both inch-pound and SI editions exist they are combined into one document separated by a slash. A trailing lowercase letter or “ε1” (e.g., A36–19) indicates an editorial revision within the year.

How AWS, ASME, and ASTM Work Together

These three standards bodies form an interlocking framework: ASTM defines what the material is; ASME specifies how to design and build pressure-critical equipment from those materials; AWS governs how to weld structural and pressure-containing components. Understanding how they reference each other is essential for reading engineering drawings, material test reports (MTRs), and inspection records.

Activity ASTM Role ASME Role AWS Role
Material procurement Defines chemistry, mechanical properties, and test requirements (e.g., A516 Gr. 70) Adopts as SA-516 Gr. 70 with supplemental ASME requirements; only SA- material may be used in Code construction Lists approved base metals in D1.1 Table 3.1 for prequalified WPSs
Filler metals No direct filler metal standard (A-prefix is structural steel) Adopts AWS A5.xx as SFA-5.xx in Section II Part C Publishes AWS A5.xx classification system (e.g., AWS A5.1 for SMAW electrodes, E7018 classification)
Welding qualification No direct role in welding qualification Section IX — WPS/PQR/WPQ using P-numbers, F-numbers, and A-numbers D1.1 Clause 4 — WPS/PQR/WQT for structural steel construction
Inspection and NDE E-series test methods (E709 for MT, E165 for PT, E94 for RT) Section V — NDE procedures referenced in all BPVC construction sections D1.1 Clause 6 — VT, RT, UT, MT, PT acceptance criteria for structural welds
Hardness testing E18 (Rockwell), E10 (Brinell), E384 (microhardness), E140 (conversions) References ASTM E-series for hardness verification of materials and welds References ASTM E-series; D1.1 limits maximum HAZ hardness for certain steels
Key distinction — ASTM vs. SA- certification: A material certified to ASTM A516 Gr. 70 cannot automatically be used in ASME Code pressure vessel construction. The mill must certify the material to SA-516 Gr. 70, which requires compliance with ASME supplemental requirements. Always verify the "SA-" (or "SB-") prefix on the material test report (MTR) before accepting material for Code work.
